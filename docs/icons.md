# Markers

## Usage
```js 
> const { Marker } = require("xmind-sdk"); 
 
> const marker = new Marker(); 
> const valueObject = marker.other("bomb"); 
> console.info(valueObject); 
{groupId: ..., markerId: ...} 
```
